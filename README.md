# Sanjyot — Web Developer & Cybersecurity Enthusiast

A modern, animated personal portfolio site built with HTML, CSS, and JavaScript.

## Features

- **Dark & luminous design** — deep navy with teal/cyan accents + subtle security green
- **Dual identity** — showcases both web development and cybersecurity interests
- **Animated gradient text** — distinctive hero typography
- **Terminal cursor blink** — subtle nod to the security/tech aesthetic
- **Mouse-tracking ambient orb** — interactive background effect
- **Scroll-reveal animations** — sections fade in as you scroll
- **Fully responsive** — desktop, tablet, and mobile layouts
- **Smooth navigation** — sticky header with mobile hamburger menu
- **Accessible** — keyboard navigation, reduced-motion support, semantic HTML
- **Blazing fast** — pure vanilla JS, no frameworks, no bloat
